# Engenharia de Prompts -- Conceitos Fundamentais

Este repositório reúne os principais conceitos e técnicas de Engenharia
de Prompts (Prompt Engineering) organizados de forma simples e direta.

## O que é Engenharia de Prompts?

É a prática de criar, ajustar e melhorar instruções enviadas a modelos
de IA para obter respostas mais úteis e precisas.

## Estrutura do Repositório

-   **01-roles**: define o papel da IA.
-   **02-params**: explica parâmetros que influenciam o modelo.
-   **03-boas_praticas**: boas práticas gerais.
-   **04-output_estruturado**: como pedir respostas em formatos
    específicos.
-   **05-condicoes**: uso de condições em prompts.
-   **06-few_shot**: técnica com exemplos no prompt.
-   **07-steps**: prompts passo a passo.
-   **08-cot**: chain of thought.
-   **09-sumarizacao**: técnicas de resumo.
-   **10-texto_complemento**: continuar e melhorar textos.
-   **11-transformacao_texto**: transformar textos.
-   **12-analise_textual**: análise de textos.
-   **requirements**: dependências.

## Objetivo

Servir como guia simples para estudar e aplicar engenharia de prompts.
